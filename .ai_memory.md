# Memoria del Proyecto — Sitio Web Personal

## Identidad
- **Propósito:** Portfolio profesional, blog técnico y cara de empresa.
- **Propietario:** Fede — desarrollador con base en Argentina.
- **Stack:** Next.js 16 (App Router, Turbopack) + JavaScript + Tailwind CSS.
- **Hosting:** Vercel (deploy automático desde GitHub).
- **Repo:** https://github.com/FedeiaTech/Portfolio-Next.js-AI-Context
- **URL:** https://fedeiatech.vercel.app

## Reglas de Oro
- Server Components por defecto; Client Components solo con interactividad.
- Todo lo que use `Math.random()` o `Date.now()` se inicializa en `useEffect`, nunca en el estado inicial del `useState` (para evitar errores de hidratación).
- Los artículos del blog se escriben en MDX con frontmatter YAML obligatorio.
- No instalar librerías sin consultarme primero.
- JavaScript vanilla — no TypeScript.
- Diseño mobile-first. Viewport 375px antes de escalar.
- Archivos en kebab-case, componentes en PascalCase.

## Bugs Resueltos
- [2026-04-03] Hydration mismatch en AmbientBackground: las nebulas usaban `Math.random()` en `useState`. Fix: inicializar vacío y llenar en `useEffect`.
- [2026-04-03] Hydration mismatch en useTime: `new Date()` difiere servidor/cliente. Fix: inicializar `useState(null)`, setear en `useEffect`, y retornar placeholder "--:--:--" mientras es null. El guard `if (!time)` debe ir ANTES de cualquier llamada a `time.toLocaleTimeString()`.
